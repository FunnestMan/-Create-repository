# Fall Indicator Mod — Forge 1.20.1
**Автор: Zot (RedStar)**

## Что делает мод
- Отслеживает высоту падения игрока в реальном времени
- Показывает точку приземления (подсвеченный блок) прямо в мире
- Рассчитывает и показывает на HUD точный урон с учётом:
  - Брони (armor + toughness)
  - Зачарования «Перо падения» (Feather Falling)
  - Эффекта «Медленное падение» (Slow Falling)
  - Приземления в воду

## Как собрать (из Forge MDK)

1. Скачай **Forge MDK 1.20.1-47.2.0** с https://files.minecraftforge.net
2. Распакуй MDK в папку
3. **Скопируй все файлы этого проекта** поверх MDK (перезапиши build.gradle, gradle.properties, settings.gradle)
4. Скопируй папку `src/` поверх папки `src/` из MDK (удали пример ExampleMod)
5. Запусти команду:
   ```
   gradlew build
   ```
6. Готовый `.jar` будет в папке `build/libs/`
